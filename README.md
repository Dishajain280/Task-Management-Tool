# ✅ Interactive Task Management Webpage

A simple yet powerful task management app built with **HTML, CSS, and JavaScript**.  
Users can manage daily tasks efficiently with **dynamic UI updates** and **data persistence via LocalStorage**.

---

## 🚀 Project Goal

Create an interactive webpage where users can:

- **Add New Tasks** instantly without page reload  
- **Mark Tasks as Completed** with strike-through styling  
- **Edit Existing Tasks** dynamically with JS events  
- **Delete Tasks** instantly from the list  
- **Filter Tasks** (All / Completed / Pending) using array filtering  
- **Persist Data** with LocalStorage  
- *(Bonus)* **Toggle Button** for theme switching (e.g., Dark/Light mode)  

---

## ✨ Features

- **Add New Tasks** → Enter task title and click “Add”  
- **Mark Completed** → Checkbox or click → strike-through effect  
- **Edit Tasks** → Inline editing with save functionality  
- **Delete Tasks** → Remove individual tasks instantly  
- **Filter Tasks** → Show All / Completed / Pending  
- **Local Storage** → Save tasks persistently across sessions  
- **Toggle Button** → Switch between light/dark themes  

---

## 🧠 Skills Used

- ✅ JavaScript DOM manipulation  
- ✅ Event handling (click, input, change)  
- ✅ Array filtering for task status
